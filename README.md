This repository contains slides and a jupyter notebook that were part of a session held at the Lorentz Center workshop **["Intra-Cluster Light: Bridging the Gap Between Theory and Observations"](https://www.lorentzcenter.nl/intra-cluster-light-bridging-the-gap-between-theory-and-observations.html)**.

The jupyter notebook contains a simple example based on the TNG API. We use the API to download fields from TNG-Cluster Group Catalog, select a cluster, then download a cutout of the cluster, and separate the ICL+BCG from the satellites.

The simplest way to run the notebook is to open it on [Google Colab](https://colab.research.google.com/github/garrethmartin/Leiden_sims_workshop/blob/main/leiden_sim_tutorial.ipynb).
